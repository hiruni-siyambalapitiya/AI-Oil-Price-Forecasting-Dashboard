## Project Overview

This project develops an AI-powered oil price forecasting dashboard that predicts future crude oil prices using historical market data and Meta Prophet.

The application provides interactive forecasting for:

- WTI Crude Oil
- Europe Brent
- Singapore Brent

The system combines machine learning, data visualization, and an interactive dashboard to support business decision-making.
## Features

- Data Cleaning
- Weekly-to-Daily Data Transformation
- Time Series Forecasting
- Meta Prophet AI Model
- Interactive Dashboard
- Confidence Interval Visualization
- Future Price Prediction
- CSV Export
